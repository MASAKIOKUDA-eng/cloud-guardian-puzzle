# クラウド・ガーディアン：セキュアパズル - プロジェクト構造

```
cloud-guardian-puzzle/
├── assets/                  # ゲームで使用する画像、音声などのアセット
│   ├── images/              # 画像ファイル
│   ├── sounds/              # 音声ファイル
│   └── styles/              # CSSスタイルシート
├── src/                     # ソースコード
│   ├── components/          # ゲームのUIコンポーネント
│   ├── core/                # ゲームのコア機能
│   ├── levels/              # ゲームレベルの定義
│   ├── models/              # データモデル
│   ├── puzzles/             # パズルのロジック
│   └── utils/               # ユーティリティ関数
├── public/                  # 静的ファイル
│   └── index.html           # メインHTMLファイル
├── tests/                   # テストコード
├── .gitignore               # Gitの除外ファイル設定
├── package.json             # npm設定ファイル
├── README.md                # プロジェクト説明
└── LICENSE                  # ライセンス情報
```
